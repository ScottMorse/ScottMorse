## About Me

Hi, I'm Scott! I'm a passionate software engineer with a background as a professional musician.

### Experience and Stack

I have broad experience as a React developer, backend developer, cloud architect, DevOps owner, and so on, for startups and the enterprise. TypeScript is my main language, and [Bun](https://bun.sh) is my runtime of choice for it, at least as my package manager, though I'm flexible with using whatever tools I need.

### My AI Use

I do use AI to help me work, but I don't use it to talk to you, so none of what is written here was generated. My profile picture is also not AI but made by the same artist that designed the `bun-workspaces` [mascot](https://bunworkspaces.com/lore).

I am not all in on AI, but I do use the tools. You'll notice I have CLAUDE.md/AGENTS.md files and the like. However, I review all code that needs to go to production and generally use a hybrid workflow where I write a good amount of code manually and regularly criticize, refactor, and trim the fat off of AI contributions, and I hold code to the same standards I have in positions I've held such as an enterprise tech lead. I also write most documentation myself, believing that I shouldn't release something I can't explain accurately myself.

I don't believe we're at a point where software can be blindly trusted to AI, at least not the open source software I work on that demands high correctness and security. I'm not building based on what I predict will be the case in a year but what the reality is today, and to me that reality demands my full attention as an engineer and the same principles I've tried to learn and follow my entire career.
